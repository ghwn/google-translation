# google-translation

This project helps you translate text by crawling Google translation page.


## Prerequisites

- [Python 3](https://www.python.org/downloads/) (Python 3.8 recommended)
- [Google Chrome](https://www.google.com/chrome/)


## How To Start

```bash
# Clone this project
$ git clone https://github.com/ghwn/google-translation.git
$ cd google-translation

# Create and activate new virtual environment
$ python3 -m venv venv
$ source venv/bin/activate

# Install dependencies
$ pip install --upgrade pip setuptools wheel
$ pip install -r requirements.txt

# Run example
$ python example.py
```
